SLS_ROBUSTER v1.5.2
-------------------

Bug Fixes

Fixed: Window no longer launches at a locked, non-resizable size. On some systems
the fixed 700x980 pixel window appeared too small to use. The window now opens at
the same default size (700x980) but can be freely resized larger by the user. The
minimum size is determined by the actual content layout so it adapts per system.
