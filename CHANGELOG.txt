SLS_ROBUSTER v1.5.3
-------------------

Visual & UI Changes

The window title bar now shows "Robuster" instead of the raw filename. Since the
app installs itself in-place on update, the filename on disk no longer changes with
each version, so the title would have been misleading.

A small version badge (e.g. "v1.5.3") now appears in the bottom-right corner of
the window, so users can always see which version is running.
